## 3) Schema / Migration Changes

- Links uniqueness widened from pair to typed edge:
  - from: `UNIQUE(from_page_id, to_page_id)`
  - to: `UNIQUE(from_page_id, to_page_id, link_type)`
- Timeline dedupe index added: `idx_timeline_dedup (page_id, date, summary)`.
- Timeline trigger/function removed (no longer touches parent page search_vector on timeline row mutations).
- Minions queue schema added:
  - `minion_jobs`
  - `minion_inbox`
  - `minion_attachments`
  - related indexes + NOTIFY trigger `minion_job_notify`.
- RLS section now enables RLS on `minion_jobs` in addition to prior tables.
